🛡️ Fall / Danger Detection — single VLM static demo

A Qwen3.5-2B vision-language model that flags falls, a person down, or distress from a short strip of video frames — trained mostly on synthetic 3D data plus real in-the-wild fall footage.

⚡ This is a static Space (no backend GPU), so it can't run the 2B model live. The example verdicts below are the deployed model's real outputs, pre-computed. To analyze your own clips, run the model directly.
